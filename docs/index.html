hr-w

HR System - نظام الموارد البشرية المبسط

Bilingual (Arabic / English) HR system built with Node.js + Express + React + Sequelize + JWT.

Stack

Layer Technology
BackendNode.js 20, Express 5, Sequelize
DatabaseSQLite / MySQL / MariaDB / Postgres (via .env)
AuthJWT (Access + Refresh tokens)
FrontendReact 18, Vite, React Router v6
i18ni18next (AR <-> EN, RTL support)
StateRedux Toolkit
ContainerDocker + Docker Compose
ProxyNginx (production)

Quick Start (Development)

# 1. Copy env files
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env

# 2. Start all services
docker compose up --build

# Backend  -> http://localhost:5000
# Frontend -> http://localhost:3000

Production Build

docker compose -f docker-compose.prod.yml up --build -d
# Nginx serves SPA on :80 and proxies /api -> backend:5000

Database Seeding (Optional)

cd backend
npm run seed
node database/seed.super-admin.js

Login

POST /api/auth/login with { email, password } and optional company_code.

GitHub Pages hosts this static landing page only. Backend API must run on a server/VPS.
